## 💻 Projetos de Estudo

Aqui estão alguns dos projetos que desenvolvi durante meus estudos:

### Projeto 1: [Portal de Notícias](https://github.com/lvieira7/exercicio_css)

#### Tecnologias Usadas:

- **HTML:** Utilizei HTML para estruturar o conteúdo da página, incluindo cabeçalho, seções de notícias e rodapé.
- **CSS:** Utilizei CSS para estilizar a página e torná-la visualmente atraente. Isso incluiu a definição de cores, fontes, layout e estilos para os elementos da página.

#### Principais Lições Aprendidas:

- **HTML e CSS:** Reforcei meus conhecimentos em HTML e CSS, aprendendo a estruturar páginas e aplicar estilos de forma atraente.
- **Organização de Código:** Entendi a importância de manter um código organizado, com classes e IDs significativos.
- **Resolução de Problemas:** Enfrentei desafios de alinhamento e estilo, melhorando minhas habilidades de resolução de problemas.
- **Versionamento com Git:** Utilizei o Git para controlar versões e manter um histórico organizado das alterações.

Este projeto foi uma oportunidade valiosa para aplicar meus conhecimentos de desenvolvimento front-end e ganhar experiência prática na criação de páginas web atraentes e funcionais.
